## Hostings

- Works out of the box:

  - fastpic
  - geekpic
  - pixhost

- Key required:

  - freeimage
  - imageban
  - imageshack
  - imgbb
  - imgur
  - uploadcare

## Installation

```sh
pip install py-image-uploader
py-image-uploader --help
```
